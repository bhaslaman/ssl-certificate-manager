## Yayın Tarihi: Subat 2026

## Yeni Ozellikler

### 1. JKS (Java KeyStore) Destegi
- **PFX → JKS** donusumu
- **JKS → PFX** donusumu
- **PEM → JKS** donusumu
- Docker image'a OpenJDK 21 eklendi
- `keytool` CLI kullanilarak donusumler yapiliyor

### 2. URL SSL Kontrolu
- Herhangi bir web sitesinin SSL sertifikasini kontrol etme
- Sertifika detaylari: subject, issuer, validity, SANs, fingerprints
- SSRF korunasi: ozel IP adresleri engellendi (10.x, 192.168.x, 127.x, 172.16-31.x)
- DNS bilgisi opsiyonu
- Yeni endpoint: `POST /api/check/url`

### 3. Tema Destegi (Koyu/Acik)
- localStorage ile tema tercihi saklaniyor
- Sayfa yuklenmeden once tema uygulanir (flash onleme)
- Sidebar'da tema degistirme butonu

### 4. SSL Yasam Dongusu Sayfasi
- Gorsel akis diagrami: KEY → CSR → CA → CERT → EXPORT
- Her adim tiklanabilir (ilgili sayfaya yonlendirir)
- i18n destegi (TR/EN)

### 5. Dokumantasyon Sayfasi
- Sertifika formatlari aciklamalari (PEM, DER, PFX, P7B, JKS, CSR)
- Donusum matrisi tablosu
- API endpoint listesi

### 6. Guncelleme Kontrolu
- GitHub API uzerinden versiyon kontrolu
- Manuel kontrol butonu (sidebar'da)
- Otomatik kontrol sayfa yuklendiginde
- Yeni versiyon varsa bildirim ve release sayfasina yonlendirme

## API Degisiklikleri

### Yeni Endpointler
```
POST /api/check/url              - URL SSL kontrolu
POST /api/convert/pfx-to-jks     - PFX'ten JKS'e donusum
POST /api/convert/jks-to-pfx     - JKS'ten PFX'e donusum
POST /api/convert/pem-to-jks     - PEM'den JKS'e donusum
GET  /api/convert/jks-aliases    - JKS alias listesi
GET  /api/system/update-check    - Guncelleme kontrolu
GET  /api/system/version         - Mevcut versiyon
```

### Guncellenen Endpointler
```
POST /api/convert/pfx-to-pem
  + key_format: "pkcs8" (default) | "traditional"
  + export_mode: "bundle" (default) | "split" (ZIP)

POST /api/convert/extract-key
  + key_format: "pkcs8" (default) | "traditional"
```

## Teknik Degisiklikler

### Dockerfile
- OpenJDK 21 eklendi (JKS destegi icin)
- JAVA_HOME environment variable

- `httpx==0.26.0` eklendi (async HTTP istekleri icin)

### Guvenlik
- SSRF korunasi: ozel IP adresleri engellendi
- Legacy OpenSSL provider etkinlestirildi (eski sertifikalar icin)

## Bilinen Sorunlar ve Notlar

1. **Docker Image Boyutu**: OpenJDK eklenmesiyle ~170MB artti
2. **JKS Donusumleri**: keytool CLI gerektiriyor (Docker icinde mevcut)
3. **SSRF Korunasi**: Sadece public IP adreslerine istek yapilabilir
